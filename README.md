# Cloud CI/CD Beginner Assignment - Ahmad Naji Awad

This project is created for the **Cloud Computing** course.

It demonstrates how to use **GitHub Actions** to automatically deploy a static website to **GitHub Pages** using a simple CI/CD pipeline.

---

##  Live Website (GitHub Pages)

https://ahmadnajy.github.io/cloud-beginner-cicd-pages-AhmadNajiAwad/

---

##  What This Project Demonstrates

- Continuous Integration (CI) using GitHub Actions
- Automatic deployment (CD) to GitHub Pages
- Workflow execution on every push to the main branch
- Automatic website updates after each successful workflow run

---

##  How to Run Locally

1. Download or clone this repository
2. Open `index.html` in your browser

---

##  How The Pipeline Works

1. When I push changes to the `main` branch,
2. GitHub Actions automatically runs the workflow.
3. It checks that required files exist.
4. Then it deploys the website to GitHub Pages.
5. The live website updates automatically after a successful run.
